Team Requirements

Notes:

1. Attending multiple days of the same event only counts as one event.

2. All requirements must be completed and documented by April 30th each year.

3. Organizing and leading an outreach event counts as a requirement for only one person except in the case of the following events, which can be counted for two people: Slice of Shoreview, Woodbury Days, MN State Fair, any new event planned.
